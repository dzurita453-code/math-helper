# Math Study Hub

Two interactive math study tools — no server needed, runs entirely in the browser.

## 📘 Algebra → Calculus III
Complete course from Algebra I through multivariable calculus:
- 6 courses, 24 lessons, 70 teaching sections
- 89 practice problems with hints and full solutions
- Progress tracking

## ⚡ Math Competition Trainer
AMC 10 to AIME level practice:
- 13 strategy lessons across 4 topics
- 55+ competition problems
- DMI-style timed rounds (3 min, 10 min, 1 hour)

## Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this folder
3. Go to **Settings → Pages → Source → Deploy from a branch → main → / (root) → Save**
4. Your site will be live at `https://yourusername.github.io/repo-name/`
